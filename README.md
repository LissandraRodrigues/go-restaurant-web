## :dart: Sobre ##

<br>

Este projeto é resultado do meu nono desafio de código no <a href="https://rocketseat.com.br/gostack">bootcamp Go Stack da Rocketseat. </a> Nele fui desafiada a colocar em prática tudo que aprendi sobre React.js junto com TypeScript, praticando o conceito de CRUD (Create, Read, Update, Delete).

## :rocket: Tecnologias ##

As seguintes tecnologias foram utilizadas nesse projeto:

- [ReactJS](https://pt-br.reactjs.org/)
- [Typescript](https://www.typescriptlang.org/)
- [CRUD](https://pt.stackoverflow.com/questions/359961/o-que-caracteriza-um-projeto-crud-b%C3%A1sico#:~:text=CRUD%20%C3%A9%20o%20acr%C3%B4nimo%20da,em%20Banco%20de%20Dados%20Relacionais.)
- [API](https://canaltech.com.br/software/o-que-e-api/#:~:text=API%20%C3%A9%20um%20conjunto%20de,Interface%20de%20Programa%C3%A7%C3%A3o%20de%20Aplicativos%22.)

## :white_check_mark: Requisitos ##

Antes de começar :checkered_flag:, você precisa ter o [Git](https://git-scm.com) e o [Node](https://nodejs.org/en/) instalados.

## :checkered_flag: Iniciando ##

```bash
# Clone este projeto:
$ git clone https://github.com/LissandraRodrigues/go-restaurant-web

# O acesse:
$ cd go-restaurant-web

# Instale as dependências:
$ yarn 

# Execute a API fake:
$ yarn json-server server.json -p 3333

# Execute o projeto:
$ yarn start

```
